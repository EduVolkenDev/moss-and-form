# Moss & Form — Astro starter

Minimal Astro project based on the artistic botanical portfolio layout.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- Main page: `src/pages/index.astro`
- Base layout: `src/layouts/BaseLayout.astro`
- Global styles: `src/styles/global.css`
- Placeholder images: `public/images/`
